Monte Carlo conditional-likelihood inference for endogenous-covariate diversification models on phylogenetic trees.

```r
devtools::install_github("franciscorichter/emphasis")
```

## The model

Species diversification is modelled as a stochastic birth-death process. Let $\lambda(s,t)$ and $\mu(s,t)$ denote the per-lineage speciation and extinction rates. We write these as linked linear predictors of endogenous covariates:

$$
\lambda(s,t) = f\!\bigl(\beta_0 + \beta_N N(t) + \beta_P P(t) + \beta_E E(s,t)\bigr)
$$

$$
\mu(s,t) = f\!\bigl(\gamma_0 + \gamma_N N(t) + \gamma_P P(t) + \gamma_E E(s,t)\bigr)
$$

where $f$ is either $f(\eta) = \max(0,\,\eta)$ (linear link) or $f(\eta) = \exp(\eta)$ (exponential link).

### Endogenous covariates

| Symbol     | Name                   | Definition                                                                |
| ---------- | ---------------------- | ------------------------------------------------------------------------- |
| $N(t)$   | Species richness       | Number of extant lineages at time $t$                                    |
| $P(t)$   | Phylogenetic diversity | Total pendant branch length: $P(t) = \sum_{s \in \mathcal{L}(t)} E(s,t)$ |
| $E(s,t)$ | Pendant age            | Time since lineage $s$ last diverged                                     |

These covariates are *endogenous*: they are generated by the diversification process itself, not externally observed. Each encodes a different hypothesis about what regulates diversification.

### Model classes

Setting different coefficients to zero recovers four core model classes, each expressing a distinct biological hypothesis:

**Constant-rate (CR)** -- `model = "cr"`, parameters $(\beta_0, \gamma_0)$

All covariate coefficients zero. Speciation and extinction proceed at constant rates, independent of clade state.

**Diversity-dependent (DD)** -- `model = "dd"`, parameters $(\beta_0, \beta_N, \gamma_0, \gamma_N)$

Rates depend on species richness $N(t)$, formalising ecological limits and niche saturation. Typically $\beta_N < 0$ (speciation declines with richness).

**Phylogenetic-diversity-dependent (PD)** -- `model = "pd"`, parameters $(\beta_0, \beta_P, \gamma_0, \gamma_P)$

Rates depend on total pendant branch length $P(t)$. Where DD counts *how many* lineages exist, PD measures *how divergent* they are.

**Evolutionary pendant (EP)** -- `model = "ep"`, parameters $(\beta_0, \beta_E, \gamma_0, \gamma_E)$

Rates depend on the lineage-specific pendant age $E(s,t)$: the time since lineage $s$ last speciated. This is the only model where rates vary *across lineages at a given time*.

| Shortcut | Active covariates | Parameters                                 | Hypothesis                         |
| -------- | ----------------- | ------------------------------------------ | ---------------------------------- |
| `"cr"` | none              | $(\beta_0, \gamma_0)$                    | No state-dependent regulation      |
| `"dd"` | $N$             | $(\beta_0, \beta_N, \gamma_0, \gamma_N)$ | Ecological limits via richness     |
| `"pd"` | $P$             | $(\beta_0, \beta_P, \gamma_0, \gamma_P)$ | Regulation by evolutionary history |
| `"ep"` | $E$             | $(\beta_0, \beta_E, \gamma_0, \gamma_E)$ | Lineage-specific age effects       |

Combinations activate multiple covariates: `~ N + PD` gives 6 parameters, `~ N + PD + EP` gives 8.

## Inference framework

The marginal likelihood of a reconstructed (extant-only) phylogeny involves integrating over all possible extinction histories:

$$
L_\theta(y_{\text{obs}}) = \int f_\theta(y_{\text{obs}}, z)\,dz
$$

Importance sampling with proposal $q(z \mid \theta, y_{\text{obs}})$ gives the unbiased estimator:

$$
\hat{L}_M(\theta) = \frac{1}{M}\sum_{m=1}^{M} \frac{f_\theta(y_{\text{obs}}, z^{(m)})}{q(z^{(m)} \mid \theta, y_{\text{obs}})}, \quad z^{(m)} \sim q
$$

For survival conditioning, a GAM emulator $\hat{P}_\theta(C)$ is trained over parameter space via forward simulation, yielding conditioned log-likelihoods:

$$
\ell_{\text{cond}}(\theta) = \ell(\theta) - \log P_\theta(C)
$$

Three estimation methods are available:

| Method     | Strategy                             | Use case                                         |
| ---------- | ------------------------------------ | ------------------------------------------------ |
| `"cem"`  | Cross-entropy method (global search) | Default starting point, no initial values needed |
| `"mcem"` | Monte Carlo EM (local refinement)    | Polish CEM results from a warm start             |
| `"gam"`  | GAM surface over parameter grid      | One-shot, non-iterative                          |

## Quick reference

```r
# Simulate
sim <- simulate_tree(pars, model, max_t, link = "linear")

# Estimate
fit <- estimate_rates(tree, method = "cem", model, link,
                      control = list(lower_bound, upper_bound, ...))
fit$pars; fit$loglik; fit$AIC

# Compare
compare_models(CR = fit_cr, DD = fit_dd, PD = fit_pd)

# Condition on survival
gam_surv <- train_GAM(sims, pars_mat, model)
fit_cond <- estimate_rates(tree, method, model, cond = gam_surv, ...)
```
