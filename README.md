## The model

Species diversification is modelled as a stochastic birth-death process. Let $\lambda(s,t)$ and $\mu(s,t)$ denote the per-lineage speciation and extinction rates. We write these as linked linear predictors of endogenous covariates:

$$\lambda(s,t) = f\!\bigl(\beta_0 + \beta_N N(t) + \beta_P P(t) + \beta_E E(s,t)\bigr)$$

$$\mu(s,t) = f\!\bigl(\gamma_0 + \gamma_N N(t) + \gamma_P P(t) + \gamma_E E(s,t)\bigr)$$

where $f$ is either $f(\eta) = \max(0,\,\eta)$ (linear link) or $f(\eta) = \exp(\eta)$ (exponential link).

### Endogenous covariates

| Symbol | Name | Definition |
|--------|------|------------|
| $N(t)$ | Species richness | Number of extant lineages at time $t$ |
| $P(t)$ | Phylogenetic diversity | Total pendant branch length: $P(t) = \sum_{s \in \mathcal{L}(t)} E(s,t)$ |
| $E(s,t)$ | Pendant age | Time since lineage $s$ last diverged |

These covariates are *endogenous*: they are generated by the diversification process itself, not externally observed. Each encodes a different hypothesis about what regulates diversification.

### Model classes

Setting different coefficients to zero recovers four core model classes, each expressing a distinct biological hypothesis:

**Constant-rate (CR)** — `model = "cr"`, parameters $(\beta_0, \gamma_0)$

All covariate coefficients zero. Speciation and extinction proceed at constant rates, independent of clade state. The CR model represents a null hypothesis: the reconstructed phylogeny carries no signal that diversification is regulated by the current state of the clade. Despite its simplicity, it serves as the foundational reference for evaluating richer models (Nee et al. 1994).

**Diversity-dependent (DD)** — `model = "dd"`, parameters $(\beta_0, \beta_N, \gamma_0, \gamma_N)$

Rates depend on species richness $N(t)$, formalising ecological limits and niche saturation. As the number of co-existing lineages increases, ecological opportunity decreases: speciation slows and/or extinction accelerates (Etienne et al. 2012). This captures the adaptive radiation narrative where high early speciation reflects open niche space, followed by a slowdown as niches fill (Simpson 1953, Schluter 2000). Typically $\beta_N < 0$ (speciation declines with richness).

**Phylogenetic-diversity-dependent (PD)** — `model = "pd"`, parameters $(\beta_0, \beta_P, \gamma_0, \gamma_P)$

Rates depend on total pendant branch length $P(t)$. Where DD counts *how many* lineages exist, PD measures *how divergent* they are. A clade of 10 recently-diverged species occupies less functional space than 10 deeply-diverged ones. PD models propose that accumulated evolutionary history — not just headcount — feeds back on diversification, with $P(t)$ acting as an endogenous proxy for competition intensity or niche packing (Qin et al. 2025). Because $P(t)$ responds differently than $N(t)$ to deep versus shallow branching, PD can in principle separate effects of richness from phylogenetic structure.

**Evolutionary pendant (EP)** — `model = "ep"`, parameters $(\beta_0, \beta_E, \gamma_0, \gamma_E)$

Rates depend on the lineage-specific pendant age $E(s,t)$: the time since lineage $s$ last speciated. This is the only model where rates vary *across lineages at a given time*. If $\beta_E > 0$, older lineages (those that have not speciated recently) diversify faster — accumulating divergence increases speciation propensity. If $\beta_E < 0$, recently-split lineages are more likely to split again ("speciation begets speciation"). The EP model connects to age-dependent speciation (Hagen et al. 2015) and protracted speciation (Etienne & Rosindell 2012), shifting attention from clade-level regulation to lineage-level heterogeneity.

| Shortcut | Active covariates | Parameters | Hypothesis |
|----------|-------------------|------------|------------|
| `"cr"` | none | $(\beta_0, \gamma_0)$ | No state-dependent regulation |
| `"dd"` | $N$ | $(\beta_0, \beta_N, \gamma_0, \gamma_N)$ | Ecological limits via richness |
| `"pd"` | $P$ | $(\beta_0, \beta_P, \gamma_0, \gamma_P)$ | Regulation by evolutionary history |
| `"ep"` | $E$ | $(\beta_0, \beta_E, \gamma_0, \gamma_E)$ | Lineage-specific age effects |

Combinations activate multiple covariates: `~ N + PD` gives 6 parameters, `~ N + PD + EP` gives 8.

### A note on the EP model

The EP model is qualitatively different from CR, DD, and PD. For the latter three, all lineages share the same instantaneous rates at any given time, so the total event rate is simply $N \cdot (\lambda + \mu)$ — computable in $O(1)$. For EP, each lineage $s$ has its own rate $\lambda(s,t)$, so the total rate requires summing over all $N$ alive lineages at each Gillespie step. With $O(N)$ events to build a tree, forward simulation is $O(N^2)$.

Furthermore, the current implementation uses a **mean-field approximation** during likelihood evaluation: for non-extinction nodes, $E(s,t)$ is replaced by $\bar{E}(t) = P(t)/N(t)$, the average pendant age. Since $f(\mathbb{E}[x]) \neq \mathbb{E}[f(x)]$ in general (Jensen's inequality), this introduces systematic bias when pendant ages are heterogeneous — precisely the scenario where the EP model is most informative.

> **Practical guidance:** The EP model is most reliable for small to moderate trees. For large trees ($N > 200$), prefer PD as a computationally tractable proxy that captures aggregate pendant-length effects without per-lineage rates.

## Inference framework

The marginal likelihood of a reconstructed (extant-only) phylogeny involves integrating over all possible extinction histories:

$$L_\theta(y_{\text{obs}}) = \int f_\theta(y_{\text{obs}}, z)\,dz$$

Importance sampling with proposal $q(z \mid \theta, y_{\text{obs}})$ gives the unbiased estimator:

$$\hat{L}_M(\theta) = \frac{1}{M}\sum_{m=1}^{M} \frac{f_\theta(y_{\text{obs}}, z^{(m)})}{q(z^{(m)} \mid \theta, y_{\text{obs}})}, \quad z^{(m)} \sim q$$

For survival conditioning, a GAM emulator $\hat{P}_\theta(C)$ is trained over parameter space via forward simulation, yielding conditioned log-likelihoods:

$$\ell_{\text{cond}}(\theta) = \ell(\theta) - \log P_\theta(C)$$

Three estimation methods are available:

| Method | Strategy | Use case |
|--------|----------|----------|
| `"cem"` | Cross-entropy method (global search) | Default starting point, no initial values needed |
| `"mcem"` | Monte Carlo EM (local refinement) | Polish CEM results from a warm start |
| `"gam"` | GAM surface over parameter grid | One-shot, non-iterative |

## Reproducing the paper experiments

The following experiments replicate the analyses in *Richter (2026), "Endogenous-covariate diversification models with Monte Carlo conditional likelihoods"*.

### Experiment 1: Likelihood validation (CR model, bird phylogenies)

Compare IS log-likelihood estimates to analytical references on empirical bird trees over a grid of $(\lambda, \mu)$ values.

```r
library(emphasis)
library(ape)

data(bird.orders)
data(bird.families)

trees <- list(
  orders   = bird.orders,
  families = bird.families
)

# Parameter grid
lam_seq <- seq(0.1, 0.5, length.out = 8)
mu_seq  <- seq(0.01, 0.2, length.out = 8)
grid    <- expand.grid(lambda = lam_seq, mu = mu_seq)

results <- list()
for (tree_name in names(trees)) {
  tree <- trees[[tree_name]]
  bt   <- sort(branching.times(tree), decreasing = TRUE)
  n    <- length(bt) + 1  # number of internal nodes + 1

  for (i in seq_len(nrow(grid))) {
    lam <- grid$lambda[i]
    mu  <- grid$mu[i]
    r   <- lam - mu
    a   <- mu / lam

    # Analytical log-likelihood (Nee et al. 1994, cond=0)
    t1      <- bt[1]
    loglik_analytical <- (
      4 * log(r) + 2 * r * t1 - 4 * log(lam) - 4 * log(exp(r * t1) - a) +
      sum(2 * log(r) + r * bt[-1] - log(lam) - 2 * log(exp(r * bt[-1]) - a))
    )

    # IS estimate (S augmented trees)
    is_est <- estimate_rates(bt, method = "gam", model = "cr",
      control = list(
        lower_bound = c(lam, mu), upper_bound = c(lam, mu),
        grid_points = 1, sample_size = 200
      ))

    results <- c(results, list(data.frame(
      tree = tree_name, lambda = lam, mu = mu,
      analytical = loglik_analytical, IS = is_est$loglik
    )))
  }
}

df <- do.call(rbind, results)
cat(sprintf("Correlation: r = %.4f\n", cor(df$analytical, df$IS)))

plot(df$analytical, df$IS, pch = 19, col = ifelse(df$tree == "orders", "steelblue", "darkorange"),
     xlab = "Analytical log-likelihood", ylab = "IS log-likelihood",
     main = "CR likelihood validation")
abline(0, 1, lty = 2, col = "red")
legend("topleft", legend = c("bird.orders", "bird.families"),
       col = c("steelblue", "darkorange"), pch = 19)
```

### Experiment 2: Diversity-dependent benchmark

Compare IS estimates against the analytical DD likelihood (DDD package) for both bird trees.

```r
library(emphasis)
library(ape)
library(DDD)

data(bird.orders)
data(bird.families)

trees <- list(orders = bird.orders, families = bird.families)

results <- list()
for (tree_name in names(trees)) {
  tree <- trees[[tree_name]]
  bt   <- sort(branching.times(tree), decreasing = TRUE)

  # DD parameter grid: lambda_0, beta_N, mu
  lam0_seq <- seq(0.3, 1.5, length.out = 5)
  bN_seq   <- seq(-0.03, -0.005, length.out = 5)
  mu_seq   <- seq(0.01, 0.15, length.out = 3)

  for (lam0 in lam0_seq) {
    for (bN in bN_seq) {
      for (mu in mu_seq) {
        # IS estimate (gamma_N = 0, i.e., extinction rate constant)
        pars <- c(lam0, bN, mu, 0)
        is_est <- tryCatch(
          estimate_rates(bt, method = "gam", model = "dd",
            control = list(
              lower_bound = pars, upper_bound = pars,
              grid_points = 1, sample_size = 200
            )),
          error = function(e) NULL
        )

        # Analytical DD likelihood from DDD
        K <- round((lam0 / (-bN)) - 1)  # carrying capacity
        dd_ll <- tryCatch(
          DDD::dd_loglik(pars1 = c(lam0, mu, K), pars2 = c(4, 1, 0, 0, 2),
                         brts = bt, missnumspec = 0),
          error = function(e) NA
        )

        if (!is.null(is_est) && is.finite(dd_ll)) {
          results <- c(results, list(data.frame(
            tree = tree_name, lam0 = lam0, bN = bN, mu = mu,
            analytical = dd_ll, IS = is_est$loglik
          )))
        }
      }
    }
  }
}

df <- do.call(rbind, results)
cat(sprintf("DD benchmark correlation: r = %.4f\n", cor(df$analytical, df$IS)))
```

### Experiment 3: Parameter recovery under PD model

Estimate parameters from simulated PD trees where no analytical likelihood exists.

```r
library(emphasis)

true_pars <- c(beta_0 = 0.60, beta_P = 0.005, gamma_0 = 0.15, gamma_P = -0.003)
n_rep     <- 20

set.seed(1)
trees <- vector("list", n_rep)
for (i in seq_len(n_rep)) {
  repeat {
    sim <- simulate_tree(pars = true_pars, model = "pd", max_t = 10, link = "exponential")
    if (sim$status == "done") { trees[[i]] <- sim; break }
  }
}

lb <- c(0.01, -0.1, 0, -0.1)
ub <- c(2, 0.1, 1, 0.1)

results <- vector("list", n_rep)
for (i in seq_len(n_rep)) {
  cat(sprintf("Rep %d/%d\n", i, n_rep))
  fit <- tryCatch(
    estimate_rates(trees[[i]], method = "cem", model = "pd", link = "exponential",
      control = list(lower_bound = lb, upper_bound = ub,
                     max_iter = 30, num_particles = 80, num_trees = 5)),
    error = function(e) NULL
  )
  if (!is.null(fit) && all(is.finite(fit$pars)))
    results[[i]] <- fit$pars
}

ok      <- !sapply(results, is.null)
est_mat <- do.call(rbind, results[ok])

cat("\nParameter recovery (PD model):\n")
for (j in seq_along(true_pars)) {
  bias <- mean(est_mat[, j]) - true_pars[j]
  rmse <- sqrt(mean((est_mat[, j] - true_pars[j])^2))
  cat(sprintf("  %s: true=%.4f  mean=%.4f  bias=%+.4f  rmse=%.4f\n",
              names(true_pars)[j], true_pars[j], mean(est_mat[, j]), bias, rmse))
}
```

### Experiment 4: Model selection (CR vs DD vs PD)

Simulate trees from known models, fit all candidates, and compare via AIC.

```r
library(emphasis)

set.seed(42)
n_per_model <- 10
max_t       <- 8

configs <- list(
  CR = list(pars = c(0.6, 0.15),
            lb = c(0.01, 0), ub = c(2, 1), link = "linear"),
  DD = list(pars = c(0.8, -0.02, 0.2, 0.0),
            lb = c(0.01, -0.1, 0, -0.1), ub = c(2, 0.0, 1, 0.1), link = "exponential"),
  PD = list(pars = c(0.6, 0.005, 0.15, -0.003),
            lb = c(0.01, -0.1, 0, -0.1), ub = c(2, 0.1, 1, 0.1), link = "exponential")
)
model_names <- names(configs)

# Simulate trees
all_trees   <- list()
true_models <- character()
for (m in model_names) {
  count <- 0L
  while (count < n_per_model) {
    sim <- simulate_tree(pars = configs[[m]]$pars, model = tolower(m),
                         max_t = max_t, link = configs[[m]]$link)
    if (sim$status == "done") {
      count <- count + 1L
      all_trees   <- c(all_trees, list(sim))
      true_models <- c(true_models, m)
    }
  }
}

# Fit all models to each tree, select by AIC
selected <- character(length(all_trees))
for (i in seq_along(all_trees)) {
  cat(sprintf("Tree %d/%d (true: %s)\n", i, length(all_trees), true_models[i]))
  fits <- list()
  for (m in model_names) {
    fits[[m]] <- tryCatch(
      estimate_rates(all_trees[[i]], method = "cem", model = tolower(m),
                     link = configs[[m]]$link,
                     control = list(lower_bound = configs[[m]]$lb,
                                    upper_bound = configs[[m]]$ub,
                                    max_iter = 20, num_particles = 80)),
      error = function(e) NULL
    )
  }
  aics <- sapply(fits, function(f) if (!is.null(f)) f$AIC else NA_real_)
  selected[i] <- if (any(is.finite(aics))) model_names[which.min(aics)] else NA
}

# Confusion matrix
ok   <- !is.na(selected)
conf <- table(True = true_models[ok], Selected = selected[ok])
cat("\nConfusion matrix:\n")
print(conf)
cat(sprintf("Accuracy: %.0f%%\n", 100 * sum(diag(conf)) / sum(conf)))
```

### Experiment 5: Survival conditioning via GAM

Train a survival-probability GAM and use it for conditioned inference.

```r
library(emphasis)
library(ape)

data(bird.orders)

# 1. Train survival GAM over CR parameter space
set.seed(1)
pars_mat <- cbind(beta_0  = runif(2000, 0.1, 1.0),
                  gamma_0 = runif(2000, 0.01, 0.5))
sims <- simulate_tree(pars = pars_mat, max_t = max(branching.times(bird.orders)),
                       model = "cr", max_tries = 0)
gam_surv <- train_GAM(sims$simulations, pars_mat, model = "cr")

# 2. Evaluate conditioned vs unconditioned likelihood on a grid
lam_seq <- seq(0.15, 0.6, length.out = 6)
mu_seq  <- seq(0.01, 0.2, length.out = 6)
grid    <- expand.grid(lambda = lam_seq, mu = mu_seq)
bt      <- sort(branching.times(bird.orders), decreasing = TRUE)

results <- data.frame()
for (i in seq_len(nrow(grid))) {
  pars <- c(grid$lambda[i], grid$mu[i])

  # Unconditioned
  fit_unc <- estimate_rates(bt, method = "gam", model = "cr",
    control = list(lower_bound = pars, upper_bound = pars,
                   grid_points = 1, sample_size = 200))

  # Conditioned
  fit_cond <- estimate_rates(bt, method = "gam", model = "cr", cond = gam_surv,
    control = list(lower_bound = pars, upper_bound = pars,
                   grid_points = 1, sample_size = 200))

  results <- rbind(results, data.frame(
    lambda = pars[1], mu = pars[2],
    unconditioned = fit_unc$loglik, conditioned = fit_cond$loglik
  ))
}

# The conditioned likelihood shifts the surface, penalising parameter
# regions where extinction probability is high
cat("Conditioning shifts log-likelihood by:\n")
print(summary(results$conditioned - results$unconditioned))
```
